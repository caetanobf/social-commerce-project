#TikTok Social Commerce Project

This is a social commerce analysis project inspired by TikTok Shop for beauty brands.

Goals:
- Analyze which types of content generate the most sales
- Compare the performance of micro vs. large creators
- Understand the impact of bundles and discounts
- Extract actionable insights for social commerce strategies
